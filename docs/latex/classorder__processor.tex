\doxysubsection{Detailed Description}
Main class for coordinating the order processing workflow. 


\begin{DoxyItemize}
\item This class adheres to the Single Responsibility Principle (SRP). It does not contain hardcoded calculation or payment logic, but rather coordinates the interaction between the shopping cart and the injected strategies. 
\end{DoxyItemize}
